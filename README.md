# 🧠 AI Resume Matcher API

Proyecto backend desarrollado en **FastAPI** con el objetivo de construir una API preparada para integrar inteligencia artificial en el análisis y comparación de currículums.

---

## 🚀 Qué estoy construyendo

Estoy desarrollando un backend que servirá como base para una plataforma capaz de:
- Analizar CVs automáticamente
- Comparar perfiles con ofertas laborales
- Generar scores de compatibilidad usando IA
